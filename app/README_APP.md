# App module (placeholder)

This repository contains a minimal scaffold for the AI App Builder Android project. The app module and feature modules will be fleshed out in follow-up commits.

To work with the scaffold in Android Studio:
- Open Android Studio -> Open an existing project -> select the repository root
- The project is not a full Gradle Android project yet; subsequent commits will add build files and a working app module.

This placeholder helps coordinate the multi-module layout and core code for provider wiring.
